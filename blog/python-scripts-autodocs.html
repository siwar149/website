Python scripts para automatizar documentos

Cuando se escribe documentos html se suelen repetir ciertos patrones de texto como !DOCTYPE html, la sección head... /head, etc. Con el objeto de automatizar el proceso de escritura de estas secciones se puede hacer un script en python. Esto implica que al ajecutar el script se creará un documento html con las secciones recurrentes ya escritas y uno solo tendrá que pasar a redactar la parte importante del documento: el contenido.

El codigo inferior es un script muy simple que realiza exactamente lo descrito líneas arriba.


		   #! /usr/bin/python3
                   # htmlTemplate.py - creates a basic template for html documents.
                   # USAGE: python htmlTemplate.py  [en]
                   # []: optional
    
                   import os
                   import sys
    
                   # check if extension specified is ".html"
                   if not sys.argv[1].endswith('.html'):
                   raise Exception('File extension must be ".html".')
   
                   # create folder if folder specified doesn't exists.
                   os.makedirs(os.path.dirname(sys.argv[1]), exist_ok=True)
   
                   # open and read template depending on which language is specified.
                   if sys.argv[2] == 'en':
                       t = open('/home/siwar/Documentos/python/own/plantilla_web_en.html')
                       template = t.read()
                       t.close()
                   elif sys.argv[2] == 'es':
                       t = open('/home/siwar/Documentos/python/own/plantilla_web.html')
                       template = t.read()
                       t.close()
   
                   # write contents of template to desired ".html" file.
                   htmlFile = open(sys.argv[1], 'w')
                   htmlFile.write(template)
                   htmlFile.close()
   
                   print(sys.argv[1] + ' was created.')
