Software que uso

Principio básico

La computadora es una herramienta útil para procesar texto, presentaciones, imágenes, video y realizar cálculos que, realizarlos sin ella, sería una tarea ardua. Sin embargo, el software que la mayoría de personas usa para hacer estas tareas no está diseñado para que la interferencia entre lo que uno piensa y lo que se plasma en la computadora sea la mínima posible.

No comulgo con la idea de estar sentado horas frente a la computadora haciendo una sola tarea, por lo tanto, el software que use debe contribuir a reducir al mínimo el tiempo que me toma preparar, por ejemplo, un ensayo, una presentación, una tesis de licenciatura, procesar grandes bases de datos, etc. Bajo estas premisas, decidí eliminar el sistema operativo Microsoft Windows de mi computador y la mayoría de programas con iterfaz gráfica de usuario (del inglés Graphical User Interface), como MS Word, MS Excel, R y demás.

Sistema Operativo

Uso una distribución del sistema operativo GNU/Linux llamada Artix Linux que, a su vez, está basada en Arch Linux, una de las distribuciones más usadas. La diferencia entre Artix y Arch es que el primero no utiliza Systemd como sistema de arranque (init system). Por lo general, las distribusiones basadas en Arch permiten al usuario tener control sobre prácticamente todos los programas que se instalan en su computadora.

Gestor de ventanas

Uso el gestor de ventanas dwm (Dynamic Window Manager) de suckless.org. Es un gestor de mínimo uso de recursos, pero fácil de personalizar para satisfacer los requerimientos de cada usuario.

Edición de texto y programación

Neovim: está basado en Vim y es más que un editor de texto. Las funcionalidades de Neovim pueden ser extendidas instalando plugins. Por ejemplo, en vez de utilizar un ambiente de desarrollo integrado para un solo lenguage de programación, como R studio en el caso del lenguage R, se puede instalar el plugin Nvim-R que permite compilar código R a través de Neovim.

Terminal

Alacritty: es un emulador de terminal con aceleración por GPU (Graphical Processing Unit) escrito en lenguage Rust. Sin duda, el emulador de terminal más rápido que existe hasta ahora.

Shell

Zsh: programa que permite a los usuarios del sistema operativo ejecutar comandos y además admite plugins de autocompletado
